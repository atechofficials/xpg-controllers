# XPG Controllers

This repository contains developer resources and documentation for a range of soldering station controllers known as XPG Controllers.

**Note:** XPG Controllers are designed by another developer. These soldering station controllers are not very popular or mainstream among electronics hobbyists, and they are also not well documented in the English language. This makes it difficult for tinkerers and electronics hobbyists to design their own soldering stations using these controllers.  
That is why I created this repository: to provide a space for well-structured and properly documented resources related to XPG Controllers. I have reverse-engineered one of these XPG Controllers and uploaded its schematics for development purposes. In the future, I plan to reverse-engineer more variants of XPG Controllers and upload them to this repository.

## About XPG Controllers

XPG Controllers are a range of smart soldering station controllers designed to drive soldering tips such as C470, C245, C210, T12, and 936. Using these controllers, users can build their own DIY soldering stations with modern features such as a graphical interface for soldering iron configuration, soldering handle sleep detection, and soldering tip change detection.

All XPG Controllers include a display unit that provides visual feedback and a user interface for interaction. This allows users to have precise control over their soldering iron.
